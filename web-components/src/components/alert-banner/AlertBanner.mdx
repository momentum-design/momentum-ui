# Alert Banner

The `<md-alert-banner/>` component

### Attributes

- `type` : String - renders the type of the alert-banner - `error, warning, default`.
- `message` : String - renders the text to be displayed on the `<md-alert-banner></md-alert-banner>`.
- `closable` : Boolean - sets the visibility of the close button on the `<md-alert-banner/>`.
- `show` : boolean - sets the visibility of the `<md-alert-banner></md-alert-banner>`.
- `closeAriaLabel` : String - accessibility text that labels the close button in the alert-banner when there is visible text on screen.
- `close-aria-label` : String - attribute for the property `closeAriaLabel`.
- `alertBanner-hide` : event - handles the functionality on closing of the `<md-alert-banner/>`.

### A simple usage of alert-banner

Renders an alert-banner with dynamic properties

```html
<md-alert-banner @alertBanner-hide=${(action('dispatchEvent'))} show type="warning" ?closable=${true} message="${`Alert Banner Message`}">
</md-alert-banner>
```

### Slots

#### Unnamed Slot

The Unnamed `<slot>` will render any content you pass between the `<md-alert-banner></md-alert-banner>` tags.
```html
<md-alert-banner @alertBanner-hide=${(action('dispatchEvent'))} show type="${`default`}" ?closable=${true}> 
    <div>My banner's content!</div> 
</md-alert-banner>
```
