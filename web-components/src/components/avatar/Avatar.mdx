# Avatar

The `<md-avatar/>` component

### Attributes

- `alt` : String - if src is present, the image alt is provided via this attribute.
- `src` : String - renders the src provided on the avatar.
- `label` : String - accessibility text that labels the avatar.
- `title` : String - renders the initials of the text provided via `title` attribute, on the avatar.
- `color` : String - color for the avatar can be provided using this property.
- `type` : String - depicts the status of the persona via different icons on the avatar. (typing, presenting, active, call etc...)
- `size` : Number - represents the size of the avatar.
- `icon-name` : String - name of the icon to be displayed inside the avatar. Must be a valid icon name.
- `has-notification` : Boolean - if true, will show the notification symbol on the avatar.
- `iconName` : String - name of the icon mapped to `icon-name` attribute.
- `hasNotification` : Boolean - mapped to `has-notification` attribute. 

### Styling

The wrapping container of the avatar can be reached via CSS by using the `:part()` selector. For example, in your application's stylesheet you can say:

```CSS
md-avatar::part(avatar) {
   background-color: red;
}
```

### Simple Usage of Avatar

#### Basic large avatar
Renders a large-sized avatar based on the value passed to the `size` property.

```html
<md-avatar alt="avatar" title="Alyson Hoagland Pace" size="44"></md-avatar>
<md-avatar alt="avatar" title="Alyson Hoagland Pace" size="44" color="purple"></md-avatar>
<md-avatar alt="avatar" title="Alyson Hoagland Pace" size="44" color="mint"></md-avatar>
<md-avatar alt="avatar" title="Alyson Hoagland Pace" size="44" color="slate"></md-avatar>
<md-avatar alt="avatar" title="Alyson Hoagland Pace" size="44" color="gold"></md-avatar>
<md-avatar alt="avatar" title="Alyson Hoagland Pace" size="44" color="pink"></md-avatar>
<md-avatar alt="avatar" title="Alyson Hoagland Pace" size="44" color="lime"></md-avatar>
```

#### Basic medium avatar
Renders a medium-sized avatar based on the value passed to the `size` property.

```html
<md-avatar
  alt="avatar"
  title="Kevin Hyde"
  size="52"
  src="https://static.skillshare.com/uploads/users/7330753/user-image-large.png"
></md-avatar>
```

#### Small Avatar with Custom image

Renders an avatar with custom image passed via `<img/>` tag enclosed in `<md-avatar></md-avatar>` tags.

```html
<md-avatar alt="avatar" title="Kevin Hyde" size="18">
  <img
    src="https://st2.depositphotos.com/4967775/11323/v/950/depositphotos_113235752-stock-illustration-avatar-girls-icon-vector-woman.jpg"
  />
</md-avatar>
```

#### Medium Type with Custom SVG Image

Renders an avatar with custom  SVG image passed via `<img src="${svg}/>` tag enclosed in `<md-avatar></md-avatar>` tags.

```html
<md-avatar alt="avatar" title="Kevin Hyde" size="40">
  <img width="100" height="100" src="${svg}" />
</md-avatar>
```

#### Icon Avatar with Color

Renders an avatar with icon and colour passed via `icon-name` and `colour` atrribute respectively.

```html
<md-avatar alt="avatar" title="Warning Icon" size="80" color="mint" icon-name="warning_20"></md-avatar>
```

#### Icon Avatar Status

iRenders an avatar with status `dnd` passed via `type` atrribute.

```html
<md-avatar title="Icon" size="36" type="dnd"></md-avatar>
```

#### Icon Avatar Group

Renders an avatar with `group` type passed via `type` atrribute.

```html
<md-avatar title="Icon" type="group"></md-avatar>
```

#### Avatar Self

Renders an avatar with `self` type passed via `type` atrribute.

```html
<md-avatar title="Self" type="self" size="24"></md-avatar>
<md-avatar title="Self" type="self" size="28"></md-avatar>
<md-avatar title="Self" type="self" size="36"></md-avatar>
<md-avatar title="Self" type="self" size="40"></md-avatar>
<md-avatar title="Self" type="self" size="44"></md-avatar>
<md-avatar title="Self" type="self" size="56"></md-avatar>
```

#### Icon Avatar Bot

Renders an avatar with `bot` type passed via `type` atrribute.

```html
<md-avatar type="bot" title="Tom Smith"></md-avatar>
```

#### Icon Avatar Notification

Renders an avatar with `notification` symbol passed via `has-notification` atrribute.

```html
<md-avatar has-notification type="active" size="72" title="Tom Smith"></md-avatar>
```

#### Icon Avatar Typing

Renders an avatar with `typing` status passed via `type` atrribute.

```html
<md-avatar type="typing" size="84" title="Tom Smith"></md-avatar>
```

#### Composite Avatar

Renders a composite avatar by wrapping 2 or more `<md-avatar/>` tags inside `<md-composite-avatar></md-composite-avatar>` tags.

```html
<md-composite-avatar size="72">
  <md-avatar type="typing" has-notification title="Tom Smith"></md-avatar>
  <md-avatar type="dnd" title="Alyson Hoagland Pace"></md-avatar>
</md-composite-avatar>
```
