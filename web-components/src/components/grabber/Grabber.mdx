# Grabber

The `<md-grabber/>` component

### Attributes

- `disabled` : Boolean - if true, `<md-grabber/>` is diabled.
- `checked` : Boolean - if true, `<md-favorite/>` is checked, else unchecked.
- `label`: String - aria label of unchecked grabber `<md-grabber/>`
- `checkedLabel`: String - aria label of checked grabber `<md-grabber/>`
- `grabber-toggled` : event - custom event displatched on toggle of `<md-grabber/>`.
- `grabber-hover-changed` : event - custom event displatched mouse enter and exit `<md-grabber/>`.

### A simple usage of `<md-grabber/>`

Renders `<md-grabber/>` with dynamic properties

```html
<md-grabber @grabber-toggled=${(action('grabber-toggled'))}  ?checked=${true} ?disabled=${false} label="Expand" checkedLabel="collapse">
</md-grabber>
```
